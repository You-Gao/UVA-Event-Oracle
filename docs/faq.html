Took a pretty cool class called INST 2500, where we got to hear from professors across different departments and schools in the university. I wanted to continue that tradition of learning but found finding interesting lectures between all of the sources would not be feasible for me in the time I have. So that's where this website came to be!

I used webscrapers to retrieve html from calendar sites, did some preprocessing, and sent it over to a LLM API so it could be processed into JSON files. Did a little fiddling with post-processing and churned it out to a .csv file. What you see in home is the file imported into Google and displayed from there. (Cooler and more interactive calendar coming soonTM!)

Feel free to contact me to update the sites the events are pulled from! The current sites it pulls from is in sites.json. If the site is there but still not listed, its probably me/the code wonked out. My email: djx3rn@virginia.edu
